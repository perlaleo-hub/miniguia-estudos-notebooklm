# miniguia-estudos-notebooklm
Projeto feito com uso da Inteligência Artificial como ferramenta de aprendizagem ativa para criar um Caderno Temático no NotebookLM. O tema desse caderno temático: Previdência Social - Aposentadoria Programada.

Contexto e Objetivos: abordar diversos temas previdenciários, com foco na Aposentadoria Programada. Será apresentada a base para o entendimento concreto do que é a Aposentadoria Programada e o quais os requisitos necessários para se aposentar. Ainda, terá explicado como fazer simulações de tempo de contribuição e salário de benefício/renda mensal inicial com base em parâmetros inseridos. Objetiva facilitar o entendimento da Previdência Social num assunto que é tão importante para a maioria dos cidadãos, de forma que os mesmos possam conhecer seus direitos e se planejar melhor financeiramente para a época cujo provento principal será a sua própria aposentadoria.
