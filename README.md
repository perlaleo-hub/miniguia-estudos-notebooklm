# miniguia-estudos-notebooklm
Projeto com uso da Inteligência Artificial como uma verdadeira ferramenta de aprendizagem ativa. Neste projeto prático, ocorre o alinhamento de pensamento crítico, curadoria de fontes e organização do conhecimento para criar um Caderno Temático no NotebookLM. Tema desse caderno temático: Previdência Social - Aposentadoria Programada
